# Mohammed Saleem | Full-Stack Developer & SEO Engineer Portfolio

A premium, high-performance portfolio website inspired by the **Mercedes F1 Petronas Teal** aesthetic. Built with a focus on clean UI systems, technical SEO, and responsive performance.

## 🚀 LIVE PREVIEW
[View the live version here](https://salim1117.github.io/portfolio/) *(Update with your actual GitHub Pages link once deployed)*

## 🛠️ TECHNICAL STACK
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons**: Lucide Icons, Font Awesome, DevIcons
- **Typography**: Inter (Google Fonts)
- **Design Style**: Mercedes F1 / Petronas Teal (Tiffany Blue) Color Palette

## ✨ KEY FEATURES
- **Premium Performance Theme**: Deep teal gradients with radial highlights for a high-executive feel.
- **Scroll Spy Navigation**: Dynamic active state highlighting as the user scrolls through sections.
- **Project Showcases**: Detailed project cards with technical contributions and tech stack labeling.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **ATS-Ready Content**: Structural layout designed for clarity and search engine visibility.
